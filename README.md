# swift-docker-view
<img src="https://img.shields.io/badge/license-MIT-green">  


**Dockerview** is a mac standard dock-like view

## Usage

```swift
import UIKit
import DockerView

class ViewController: UIViewController {
    let dockerView = DockerView()

    override func viewDidLoad() {
        super.viewDidLoad()
        // setup dockerView layout
    }
}
```
## Installation


### CocoaPods
 - Add `pod 'DockerView'` to your Podfile.
 - Run pod update.

## Requirements
 - iOS 13.0 or later
